🚀 Billetera Virtual API💳 Gestión financiera inteligente y eficienteUna solución Full Stack para la gestión de finanzas personales, permitiendo el registro de usuarios, control de saldos y seguimiento de transacciones en tiempo real.📖 Tabla de ContenidosCaracterísticasArquitecturaOptimización de Recursos (Low-Cost)TecnologíasConfiguración del ProyectoEndpoints de la API✨ CaracterísticasAutenticación Segura: Sistema de Login y Registro con persistencia en LocalStorage.Gestión de Saldo: Actualización dinámica de fondos tras cada transacción.Historial de Movimientos: Registro detallado de ingresos y egresos.Diseño Responsive: Interfaz adaptada a dispositivos móviles y escritorio.Despliegue Continuo (CI/CD): Integración automática mediante GitHub Actions en Vercel y Railway.🏗 ArquitecturaEl sistema opera bajo un modelo de Arquitectura Desacoplada, facilitando el mantenimiento y escalabilidad independiente del Frontend y el Backend.⚡ Optimización de Recursos (Low-Cost)Este proyecto ha sido específicamente tuneado para ejecutarse en el plan gratuito de Railway, optimizando el consumo de la JVM (Java Virtual Machine):🔧 Ajustes de Memoria (Environment Variables)VariablePropósito-XX:+UseSerialGCReduce el uso de CPU y RAM al recolectar basura en un solo hilo.-Xss256kMinimiza la memoria por cada hilo del servidor.-XX:MaxRAMPercentage=75.0Limita el uso de la memoria al 75% del contenedor, evitando reinicios.🛠 Configuración de HikariCP (application.properties)Properties# Pool de conexiones eficiente
spring.datasource.hikari.maximum-pool-size=5
spring.datasource.hikari.idle-timeout=30000
